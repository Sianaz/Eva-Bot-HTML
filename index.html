¿No te gusta el prefijo predeterminado?

Sino te gusta el prefijo predeterminado puedes usar el comando setprefix de esta forma: e!setprefix [nuevo prefix] y si todo sale bien el prefix se cambiara en tu servidor

¿Como veo la Lista de Comandos?

para ver la lista de comandos puedes usar el comando help de esta forma Eva te mandara un embed con todos sus comandos(los de desarrollador estan ocultos) y te mandara un mensaje privado con mas informacion
